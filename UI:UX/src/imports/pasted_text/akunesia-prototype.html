Buatkan prototype interaktif FULLY CLICKABLE untuk website edukasi "Akuanesia" — platform virtual lab berbasis machine learning untuk mahasiswa kelautan dan perikanan Indonesia, yang memungkinkan eksplorasi kecocokan spesies ikan dengan kondisi perairan berdasarkan data kualitas air historis.
